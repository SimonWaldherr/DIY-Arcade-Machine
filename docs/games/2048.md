# 2048 (`2048`)

## Gameplay

- Slide the 4x4 tile grid in one direction at a time.
- Equal tiles merge into the next value and add to the score.
- Reach the 2048 tile or keep playing until no moves remain.

## Technical details

| Language-agnostic design | Python implementation in this repository |
| --- | --- |
| A move compacts each row or column, merges equal neighbors once, then fills empty cells. | `Game2048._move()` builds packed lines and uses `_compress_line()` for merge scoring. |
| A new tile appears after every successful move. | `_spawn_random()` adds a 2 or occasional 4 to a random empty cell. |
| The compact LED UI needs short value labels. | Tiles render as single-character power levels instead of full numbers. |
